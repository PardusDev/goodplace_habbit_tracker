<h1 align="center">GoodPlace</h1>
<h3 align="center">GoodPlace is the most intuitive and user-friendly habit tracking app, powered by AI. By consistently tracking your habits, you'll be able to stay on track with your goals and achieve lasting personal growth</h3>

<br><br>

## Project Overview

This project was developed for Milu's intern training program called Grow Together. In this app, you can track your habits. The Kartal package was used to keep the app consistent, and the Provider package was used for state management. Firebase was used for user registration and database tasks. In some cases, APIs were used, and the Dio package was used for API tasks.

## Getting Started

To get a local copy up and running follow these simple steps.

1. Clone the repository
2. Open the project in Android Studio
3. Install dependencies
4. Run the project

## Used Packages
- **[firebase_core](https://pub.dev/packages/firebase_core)**: Used for initializing Firebase in the application, ensuring a stable connection with Firebase services.
- **[firebase_auth](https://pub.dev/packages/firebase_auth)**: Handles user registration and authentication, providing secure sign-in options.
- **[google_sign_in](https://pub.dev/packages/google_sign_in)**: Integrates Google Sign-In, allowing users to sign in with their Google accounts easily.
- **[cloud_firestore](https://pub.dev/packages/cloud_firestore)**: Manages database tasks, such as storing and retrieving habit data from Firebase Firestore.
- **[provider](https://pub.dev/packages/provider)**: Facilitates state management across the app, ensuring data consistency and smooth user interactions.
- **[shared_preferences](https://pub.dev/packages/shared_preferences)**: Stores user preferences and data locally, enabling persistent settings and configurations.
- **[dio](https://pub.dev/packages/dio)**: Utilized for making API requests, providing efficient and robust handling of network operations.
- **[kartal](https://pub.dev/packages/kartal)**: Ensures a consistent UI/UX throughout the app by offering utility functions and extensions.
- **[animated_text_kit](https://pub.dev/packages/animated_text_kit)**: Adds engaging animated text effects, enhancing the visual appeal of the app.
- **[package_info_plus](https://pub.dev/packages/package_info_plus)**: Retrieves app-specific information such as version and build number.
- **[shimmer](https://pub.dev/packages/shimmer)**: Implements shimmer effects to indicate loading states, improving user experience during data fetching.
- **[get_it](https://pub.dev/packages/get_it)**: Provides dependency injection, making the app more modular and easier to manage.
- **[table_calendar](https://pub.dev/packages/table_calendar)**: Displays a calendar widget on the home page, allowing users to view and manage their habits on specific dates.
- **[lottie](https://pub.dev/packages/lottie)**: Integrates Lottie animations, bringing dynamic and visually appealing animations to the app.
- **[image_picker](https://pub.dev/packages/image_picker)**: Allows users to pick images from their gallery or take new photos, used in various features within the app.
- **[uuid](https://pub.dev/packages/uuid)**: Generates unique IDs for different entities within the app, ensuring data uniqueness.
- **[permission_handler](https://pub.dev/packages/permission_handler)**: Manages permissions, such as accessing the camera or location, ensuring that the app functions smoothly and securely.
- **[cached_network_image](https://pub.dev/packages/cached_network_image)**: Caches images fetched from the network, reducing load times and enhancing user experience.
- **[flutter_widget_from_html_core](https://pub.dev/packages/flutter_widget_from_html_core)**: Renders HTML content within the app, allowing for rich text and media presentations.
- **[firebase_messaging](https://pub.dev/packages/firebase_messaging)**: Handles push notifications, enabling the app to send reminders and important updates to users.
- **[flutter_local_notifications](https://pub.dev/packages/flutter_local_notifications)**: Implements local notifications for daily reminders and habit tracking prompts.
- **[timezone](https://pub.dev/packages/timezone)**: Manages time zone conversions, ensuring that reminders and notifications are triggered at the correct local time.
